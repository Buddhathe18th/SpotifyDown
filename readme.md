# SpotifyDown

A simple Python application to download Spotify songs

---

### Setup

Make sure you have:
- Python 3.8 or newer  
- `pip` (Python package manager)  

#### Important

Make sure you have the Spotipy API keys in `.env`, including `SPOTIPY_CLIENT_ID, SPOTIPY_CLIENT_SECRET, SPOTIPY_REDIRECT_URI`

Try and run 
````
git clone https://github.com/Buddhathe18th/SpotifyDown.git
cd SpotifyDown
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
````
